## お客さんの使い方

### どんなアプリ？

Vacanseat はイマスグ空いているお店を探したいアナタのためのアプリで、近くで空いているお店(カフェ、レストラン、バー)をその場で検索、予約ができるアプリです！




## オーナーの使い方

### どんなアプリ？

Vacanseat はお客さんに現在の空席情報を伝えることができ、安全に予約も受け付けることができるアプリです！
